# 🌙  LunaCard - Personal Linktree Alternative

My self-hosted, pixel-style link hub with seasonal particles, dark aesthetic, and Japanese-inspired design.

## ✨  Features

- 🎨  Custom pixel/indie web aesthetic
- 🌸  Seasonal floating particles (cherry blossoms, snowflakes, etc.)
- 📱  Fully responsive (mobile + desktop)
- 🚀  Self-hosted with Cloudflare Tunnel
- 🎯  Easy to customize profile and links

## 🛠️  Tech Stack

- React 18 + TypeScript
- Vite
- React Icons
- CSS3 (animations, pixel effects)

## 🚀  Quick Start

### Development

```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
server -s dist -l 6969
```

## 📝  Configuration
Edit src/app.tsx to customize:
- PROFILE - your name, bio and avatar
- LINKS - all your social/media links
